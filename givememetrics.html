What is GiveMe Metrics?

GiveMe Metrics is a conceptual framework, created from the tool's integration able to extract historical software data. The tools that make up the framework GiveMe Metrics (GMM) was obtained in a systematic mapping process in which criteria were set so that a tool could be selected to compose the framework.
To learn more, read the article: GiveMe Metrics
